# Privacy Policy / 隐私权政策

**Last updated: May 2025**

## 1. Information Collection / 信息收集

"X Comment Blocker" (the "Extension") does NOT collect, transmit, distribute, or sell any of your personal information, browsing history, or user data.

本扩展程序（"X 内容过滤器"）不收集、不传输、不分发、也不出售您的任何个人信息、浏览历史或用户数据。

## 2. Local Storage / 本地存储

All data generated by the Extension, including your custom blocked keywords and settings, is stored strictly locally on your device using the `chrome.storage.local` API. This data never leaves your browser.

本扩展程序产生的所有数据（包括您的自定义屏蔽词和设置开关）均使用 `chrome.storage.local` API 严格存储在您的本地设备上。这些数据永远不会离开您的浏览器。

## 3. Web Requests & Third-Party Services / 网络请求与第三方服务

1. **Cloud Keywords / 云端词库**: The Extension makes secure requests to `api.github.com` to download the latest public cloud-based keyword list. No user identification is sent during this process.
   (本扩展程序向 `api.github.com` 发出安全请求以获取公开云端词库。此过程不会发送任何用户身份标识。)

2. **X (Twitter) API Integration / X (Twitter) 接口集成**: If you use the "Block User" feature in the history panel, the Extension will read your local `ct0` cookie for `x.com` and send requests to X's official API on your behalf to block or unblock the specified account. These cookies and requests are strictly used for this functionality and are NEVER transmitted to any third-party servers.
   (当您使用历史记录中的“拉黑”功能时，扩展程序会读取您本地 `x.com` 的 `ct0` Cookie，并代表您向 X 官方接口发送拉黑或解除拉黑的请求。这些 Cookie 及请求仅严格用于此功能。)

## 4. Contact / 联系方式

If you have any questions, please open an issue in this GitHub repository.

如果您有任何问题，请在此 GitHub 仓库中提交 Issue。
